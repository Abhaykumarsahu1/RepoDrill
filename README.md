# RepoDrill

A high-performance codebase auditing tool built in Rust.

## Overview

RepoDrill is a language-agnostic repository auditing tool that helps developers identify technical debt, documentation gaps, code quality issues, and potential security risks.

The tool scans local codebases and generates actionable insights through a fast and interactive terminal experience.

## Features

- Repository-wide code scanning
- TODO/FIXME detection
- Documentation coverage metrics
- Potential secret/API key detection
- Large file identification
- Code health scoring
- Interactive TUI using Ratatui
- Async file processing using Tokio

## Tech Stack

- Rust
- Tokio
- Ratatui
- Regex
- Crossterm
